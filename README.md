This project generates synthetic monthly sales data, fits a best-fit linear regression line using NumPy, and plots the results using Matplotlib.
It also includes CircleCI integration and automated unit testing.

Features
Generates synthetic monthly sales data
Computes a line of best fit using least squares
Predicts values based on the fitted line
Saves a plot of the data and regression line
Includes CircleCI configuration for continuous integration
Automated unit tests to ensure reliable functionality
